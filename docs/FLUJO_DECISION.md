# Flujo de Decisión y Justificación de Umbrales

Este documento debe detallar la lógica utilizada para clasificar el estado del equipo.

## Umbrales Propuestos (Plantilla)
* **Temperatura:** Normal (<50°C), Preventivo (50-75°C), Falla (>75°C).
* **Vibración:** Normal (<2g), Preventivo (2-5g), Falla (>5g).
* **Intensidad de Uso:** Normal (<60%), Preventivo (60-85%), Falla (>85%).

*(A rellenar con normas y estándares justificados)*
